# n8n Workflow Pack

This package is for importing governed workflow examples and getting one safe automation run working first.

Included:
- `workflows/SCBE_X_Merch_Autopilot.workflow.json`
- `workflows/connectors.n8n.json`
- `environment-variable-map.md`
- `connector-notes.md`
- `validation-checklist.md`

Success check:
- one workflow imports cleanly
- required variables are mapped
- one safe run completes

Not included:
- your external credentials
- managed hosting
- automatic production rollout
