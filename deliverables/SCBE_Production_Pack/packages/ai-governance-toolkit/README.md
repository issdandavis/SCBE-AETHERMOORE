# AI Governance Toolkit

This package is for adding a small practical governance layer before you try to deploy a bigger system.

Included:
- `decision-record-template.md`
- `threshold-worksheet.md`
- `pilot-checklist.md`
- `review-notes-template.md`

Use it for:
- documenting allow/quarantine/deny decisions
- setting first safe thresholds
- running a pilot on one workflow

Success check:
- one completed threshold worksheet
- one completed decision record tied to a real workflow

Not included:
- full SCBE architecture internals
- hidden weighting values
- managed deployment
