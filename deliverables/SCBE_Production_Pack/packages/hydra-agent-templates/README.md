# HYDRA Agent Templates

This package is for small governed swarms that need clear role ownership and cleaner handoffs.

Included:
- `role-templates.md`
- `packet-example.md`
- `launch-checklist.md`
- `review-checklist.md`

Success check:
- one swarm launched with explicit role ownership
- one clean packet handoff produced

Not included:
- a full autonomous swarm runtime
- infinite role variations
- managed orchestration
