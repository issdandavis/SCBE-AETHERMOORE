# Packet Example

## Mission

- Goal: Verify a public launch page and fix one broken buyer path.

## Ownership

- Explorer: confirm live route and broken links
- Worker: patch the site files
- Reviewer: check that the route and support links agree

## Evidence expected

- exact file paths changed
- exact route checked
- one decision on whether the path is fixed

## Completion rule

- broken route resolved or explicitly deferred with reason
