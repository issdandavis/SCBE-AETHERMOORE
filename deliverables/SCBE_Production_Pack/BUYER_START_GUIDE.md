# SCBE Buyer Start Guide

## First ten minutes

1. Save the receipt email.
2. Open the package folder that matches what you bought.
3. Read that package `README.md` before opening any other project docs.
4. Confirm the expected files are present.
5. Complete one small success check.

## What is not included by default

- managed deployment
- hosted keys unless the specific product says it needs them
- private internal threshold values
- custom integration work
