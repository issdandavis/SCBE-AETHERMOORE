# Storyboard Tactics Reference — Manhwa/Webtoon Production

## Panel Size = Emotional Weight
- **Largest panel = most important moment** — longest dwell time during scroll
- Small/narrow panels compress time and speed reading
- **3 narrow → 1 wide** pattern: tension, tension, tension, payoff
- Full-width reserved for: character reveals, climaxes, establishing shots
- Maximum 2 panels visible at once on mobile

## Scroll Gaps as Pacing
| Gap Size | Meaning |
|---|---|
| 50-100px | Quick continuation, same beat |
| 150-300px | Scene breath, small pause |
| 400px+ | Major dramatic pause, revelation settling |
| 600-1000px | Scene/location change, act break |

**The gap IS the time.** More scroll = more elapsed story-time.

## Shot Progression (Film Grammar)
1. **Wide establishing** (full-width) → where are we?
2. **Medium shot** → who is here?
3. **Close-up** → what are they feeling?
4. **Reaction** → how does the other respond?
5. **Reverse** (zoom out to close scene)

## Camera Angles
| Angle | Effect | Use For |
|---|---|---|
| Eye-level | Neutral, relatable | Default dialogue |
| Low angle (up) | Power, dominance | Reveals, intimidation |
| High angle (down) | Vulnerability, smallness | Defeat, being judged |
| Bird's eye | God-view, detachment | Establishing geography |
| Dutch angle | Unease, wrongness | Horror, instability |
| Over-shoulder | Dialogue intimacy | Conversations |
| Extreme close-up | Raw emotion | Eyes, hands, key objects |
| Worm's eye | Extreme awe | Giant creatures, towers |

**Solo Leveling specifics**: extreme low angles on power moments, dynamic foreshortening, complementary color shadows, chromatic aberration on impact.

## Color Storytelling
- **Palette shifts between scenes** signal location change before content
- **Desaturation = flashback/memory** (nearly universal)
- **Warm colors**: energy, comfort, safety, passion
- **Cool colors**: calm, melancholy, mystery, danger
- **Spot color**: one saturated element in muted scene = instant focal point
- Use sparingly — power comes from contrast

## Text/Lettering Rules
- **Narrator top of panel** = sets context BEFORE image
- **Narrator bottom of panel** = reflects AFTER image lands
- **Speaker on left = bubble on left**
- **Higher bubble = earlier in conversation**
- **Max 3 bubbles per panel** (WEBTOON guideline)
- **Tails never cross each other**

### When to use NO text
- Establishing shots
- Emotional peaks (dialogue would undermine)
- Beat panels (silence IS the point)
- Horror/dread
- Aftermath of climactic events

### SFX as Composition
- Korean/Japanese: SFX drawn INTO artwork, part of composition
- Korean has 3x more onomatopoeia than English
- Includes not just sounds but emotions (heartbeat) and textures (roughness)

## Panel Transitions (Scott McCloud)
| Transition | What Happens | Use |
|---|---|---|
| Moment-to-moment | Tiny time increment | Tension, subtle changes |
| Action-to-action | Cause → effect | Workhorse. Punch → hit. |
| Subject-to-subject | Different characters, same scene | Dialogue, reactions |
| Scene-to-scene | Time/location jump | "Meanwhile..." cuts |
| Aspect-to-aspect | Different details, SAME moment | Mood building, atmosphere |
| Beat panel | Minimal/empty | Dramatic pause |

**Key insight**: Manga/manhwa use significantly more aspect-to-aspect than Western comics. This is why they "feel different."

## Character Introduction Tactics
1. **Delayed reveal**: Show pieces (hand, shadow, boot) before the whole
2. **Silhouette-first**: Shape communicates threat/mystery before detail
3. **Low angle for first appearance**: Character towers, appears powerful
4. **Full-width panel**: Largest panel in episode for character intro
5. **Two-stage**: Notice them in normal panel → SEE them in splash panel
6. **Manhwa convention**: Dedicated "fashion plate" introduction panel (purely about the LOOK)

## Cliffhanger Techniques
- **Last panel = question or threat** — fundamental webtoon structure
- **Scroll gap before reveal** = the drumroll (400-1000px empty space)
- **Reveal panel = full-width, taller than normal**
- **Emotional > plot** for retention in character-driven stories
- **Strongest = both**: plot event that creates emotional crisis

## Post-Credits Stinger
- 1-3 panels AFTER the "ending," separated by large gap
- Rewards readers who scroll past apparent end
- Recontextualizes what reader just experienced
- "Someone was watching" / "consequence is coming" / "victory was premature"

## Thumb-Stop Architecture
- Plan **2-3 thumb stops per episode** — moments reader naturally pauses
- Place at: first revelation, midpoint escalation, final cliffhanger
- Each = full-width panel with high visual impact
