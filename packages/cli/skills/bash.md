# Bash / Shell Skill Card

summary: Run shell, PowerShell, Bash, or cmd work through governed SCBE receipts.
triggers: bash, shell, powershell, cmd, terminal, run

## Worksheet

- Identify the concrete command separately from the user's prose.
- Prefer `scbe x <command>` for execution so the command produces a receipt.
- If the input is prose and not a recognizable command, do not execute it as a shell token.
- Preserve stdout, stderr, exit code, duration, and next-step failure summary.
