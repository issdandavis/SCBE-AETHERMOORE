# Fetch / Retrieve Skill Card

summary: Retrieve remote or local information, then summarize what was actually fetched.
triggers: fetch, download, retrieve, lookup, look up

## Worksheet

- Identify source, query, output target, and cache policy.
- Avoid fetching secrets into logs.
- If web or remote state matters, record the source and date.
- Return a compact result plus a trace of the fetch command.
