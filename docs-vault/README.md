# Documentation Vault

This vault stores non-runtime documentation moved out of the active source tree.

## Phase 1 moves

- `notes/` -> `docs-vault/notes/`
- `references/` -> `docs-vault/references/`
- `content/articles/` -> `docs-vault/content/articles/`

## Notes

- These moves are intended to preserve build/runtime behavior while reducing root-level documentation clutter.
- Runtime/config/system assets remain in their original paths.
