## 5 Cornerstone Art Styles (with timestamps from recap video)

### 1. Standard Dialogue
- Soft cel-shading, proportional figures, soft backgrounds
- Used for: exposition, character interaction, comedy beats
- Palette: warm cream/gold (palace), context-dependent
- Example: 10min mark — butler Syde comedy scene

### 2. Ethereal/Divine
- White/gold palette, floral elements, painterly softness
- Used for: dragon realm, supernatural reveals, spiritual moments
- Completely different palette from standard scenes
- Example: 30min mark — blonde figure in white robes

### 3. Chibi/SD
- Super-deformed proportions, huge Korean SFX text
- Used for: comedy punchlines, reaction beats
- Breaks proportion rules deliberately
- Example: 90min mark — green-eyed guy comedy panel

### 4. Dark/Gritty
- Muted palette, stone textures, harder shadows
- Used for: action, medieval settings, tension
- MC outfit shifts to adventurer gear
- Example: 45min mark — stone arch setting

### 5. Infographic
- Silhouettes, crests, text boxes, breaks panel format entirely
- Used for: lore dumps, world-building exposition, character intros
- Example: 120min mark — family crest + silhouette + text overlay

## Key Techniques
- Panel framing shifts (tight crop → full body → extreme close-up)
- Korean SFX overlaid directly on panels
- Background blur for depth (sharp foreground, dreamy background)
- Sparkle/particle effects for atmosphere
- Two-character dynamic contrast (poses, expressions)
- Soft watercolor lighting (no hard shadows)
