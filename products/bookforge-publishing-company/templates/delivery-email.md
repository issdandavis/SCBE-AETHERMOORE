# Delivery Email

Subject: BookForge Publishing Kit delivery

Thanks for buying the BookForge Publishing Kit.

Attached or linked is the current kit ZIP. Start with `START_HERE.md`, then copy
one profile from `profiles/` into your book folder as `bookforge.json`.

The fastest test path is:

```powershell
pip install scbe-bookforge
bookforge build bookforge.json --blurb blurb.json
```

If PyPI is not live yet, install from the included wheel in `engine-dist/`.
