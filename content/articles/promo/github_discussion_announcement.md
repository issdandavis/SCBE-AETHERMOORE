## How the code and the book relate

This isn't a case where someone wrote a novel and then made a repo. Or built software and then wrote a tie-in. The two artifacts emerged from the same process.

The pipeline:

1. **12,596 paragraphs of AI DnD** on Everweave generated the seed corpus
2. Those logs were expanded into worldbuilding, character systems, and a magic system with formal rules
3. The formal rules became the **6 Sacred Tongues** — constructed languages mapped to neurotransmitters, each with 256 tokens and golden-ratio weighting
4. The Sacred Tongues became the **tokenizer seed** for SCBE-AETHERMOORE
5. The security model (Poincare hyperbolic geometry, 14-layer pipeline, exponential cost scaling) became the **physics of the fictional world**
6. The novel tells the story of someone encountering that world for the first time
