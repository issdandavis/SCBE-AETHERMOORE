Evidence boundary

Claims stay attached to commands.

SCBE does not need inflated language. The useful claim is that it can preserve, classify, and explain state through transformations better than a single-field baseline in the tests it actually runs.

105/105 Longform chain integrity benchmark: structural and semantic attack cases.
3/3 Compound decomposition/recomposition local benchmark with RDKit available.
14/14 Full-system artifact readiness matrix; pass/partial status remains explicit.
12/12 Vector-field navigation ensemble-beam benchmark in the current proof lane.

These are local and project-scoped results unless a page explicitly names a third-party leaderboard. Enterprise buyers can request the exact commands, hashes, and artifact bundle used for any claim.
