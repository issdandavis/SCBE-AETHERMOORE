## Key Decisions

1. **Dual layer architecture**: A project hook auto-generates simplified M6 versions of dressing.py and composition.py. Full geometric implementations live in `*_geometric.py` files. Both coexist.

2. **Spin coherence fix**: L10 uses local phasors (tongue's own phase ± small perturbations) instead of all 6 tongue phases, since the 6 tongues are uniformly distributed by design and would always give near-zero spin coherence.

3. **Phase deviation fix**: L12 uses `1 - spin_score` as phase deviation instead of raw phase comparison, which was comparing incompatible scales.

4. **Poincare clamping**: Added `min(t, 1.0 - 1e-7)` to `poincare_project()` to prevent boundary landing at exactly 1.0 due to float precision.
