About this page

Reads /v1/polly/stats?date=YYYY-MM-DD for today + the previous 6 days. Counts come from listing per-day directories on the HF dataset; no contents are exposed. The endpoint is rate-limited (60/min, feedback bucket) and cached 60 s per Vercel instance. noindex so search engines don't index the page.
