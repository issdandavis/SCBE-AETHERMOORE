Example output

The package is built for decisions.

report.md

# Findings

The workflow has useful automation, but tool calls need approval boundaries before production use.

## Highest risk
External source ingestion can influence the agent before the execution gate.

## Three fixes
1. Separate retrieval from execution.
2. Add route receipts to tool calls.
3. Require human approval for write actions.

Safety boundary

No secrets. No regulated production data.

Send redacted examples, public links, screenshots, notes, or diagrams. Do not send API keys, private keys, customer records, bank data, medical records, export-controlled material, classified material, or regulated production data.
