What is strong enough to show now

The defensible public story is narrow: SCBE preserves useful task completion while attaching governance decisions to actions, blocks clear adversarial shell actions, and detects ledger tampering in longform workflows. Local fixtures are labeled as local fixtures.

  • Terminal-Bench core parity: 13/13 SCBE and 13/13 oracle on the current neutral packet.
  • Governance separation: reverse shell, disk wipe, and bulk destructive delete land in DENY.
  • Longform integrity: byte flip, hash edit, event insert/delete/swap, recompute drift, truncation, and cold resume are covered.
  • Hydra conservation: multi-role project obligations are retained better than simple local baselines.

The next step is one unchanged external harness run. Terminal-Bench 2.x is first because it is closest to the current agent surface. AgentDojo and tau-bench follow because they test the governance problem directly: tool attacks, policy compliance, multi-turn state, and action safety.
