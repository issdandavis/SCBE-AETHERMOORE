## Task 1 - Program Kickoff, Baseline Definition, and Data Architecture

**Lead**: SCBE / Issac  
**Support**: Hoags limited to Sub Background IP review if needed  
**Period**: Month 0-1  
**Milestone payment**: `$84,000`

### Objective

Establish the Phase I mathematical framework, science subdomains, foundation models, baseline protocol, personnel assignments, initial evaluation rubric, and IV&V artifact structure.

### Work Description

SCBE will present the selected mathematical, information-theoretic, and systems-theory basis for the TA1 framework. The framework treats agents as composed operators over communication streams and latent-state readouts, with protocol progress measured through hyperbolic geometry, protocol graph structure, and proposer-added metrics.

SCBE will define the initial task families, foundation models, latent-space access plan, handcrafted baseline protocol, governance pipeline, data schema, and initial compute plan. The kickoff will include the IV&V bundle schema for communication streams, latent states, governance event logs, and metric summaries.

### Deliverables

- Kickoff briefing.
- Foundation model and latent-space access plan.
- Baseline protocol description.
- Initial TA1 rubric mapping.
- Personnel and level-of-effort confirmation.
- Draft IV&V data bundle schema.
- Initial compute-by-month plan.

### Success Criteria

- DARPA receives a concrete Phase I plan with all named personnel assigned.
- Baseline protocol and task families are defined.
- IV&V data-sharing structure is specified.

## Task 2 - Initial Mathematical Framework and Early Baseline Experiments

**Lead**: SCBE / Issac  
**Support**: Hoags optional context only if verified artifacts are available  
**Period**: Month 1-3  
**Milestone payment**: `$109,000`

### Objective

Produce the first technical report on early successes/failures, baseline model behavior, and initial mathematical-framework implementation.

### Work Description

SCBE will implement the initial operator-chain instrumentation for selected fixed pretrained agents and run early baseline tasks. ### Deliverables

- Personnel level-of-effort confirmation.
- Updated risk register.

### Success Criteria

- Initial quantitative evidence exists for the baseline protocol.

- Framework description is sufficiently concrete for M6 expansion.
