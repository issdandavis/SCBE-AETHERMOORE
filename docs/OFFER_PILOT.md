# SCBE Pilot Offer (2 Weeks)

## What You Get

- Governed browser automation for real operations tasks
- Multi-agent execution with `ALLOW/QUARANTINE/DENY/NOISE` decisions
- Verification scorecards and replay traces
- Training growth report from your own data

## Deliverables

- `artifacts/evidence_packs/<run_id>/scorecard.json`
- `artifacts/evidence_packs/<run_id>/README.md`
- Decision records and traces from governed runs
- Training metrics + growth confirmation report

## Timeline

1. Day 1-2: Setup and baseline run
2. Day 3-7: Workflow wiring (Asana/n8n/browser jobs)
3. Day 8-12: Hardening and monitoring
4. Day 13-14: Buyer-ready evidence handoff

## Success Criteria (Pass/Fail)

- Browser tasks execute with verification score >= 0.90 for approved flows
- Every high-risk action is capability-gated
- Decision records generated for every run
- Training run shows measurable growth (`val_accuracy_gain > 0` or `val_loss_drop > 0`)

## Input Needed From Client

- Target workflows (3 to 10 tasks)
- Required systems (Asana, Notion, browser targets)
- Access keys/tokens in environment variables

## Commercial Structure (Simple)

- Fixed implementation fee for pilot
- Optional monthly support retainer after pilot
- Optional expansion phase for multi-team rollout
