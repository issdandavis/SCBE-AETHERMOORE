Delivery proof

The buyer packets are built by python scripts/package_products.py into the ignored local lane products/packaged/. That keeps paid ZIPs out of the public website while still letting the repo verify that every source file exists and every package can be rebuilt.

For production, configure SCBE_TOOLKIT_DOWNLOAD_URL and SCBE_TRAINING_VAULT_DOWNLOAD_URL to point at buyer-only package links. Without those variables, the backend falls back to the public GitHub release URL for smoke testing.
