# docs/ — moved

A high-value subset of this directory was extracted from the monolith and now lives in its own repo:

**→ https://github.com/issdandavis/scbe-docs-archive**

The `scbe-docs-archive` repo contains the curated subset of the former `docs/` tree — the documents worth preserving as a readable archive. The remainder (drafts, stale notes, autogenerated reports, and supersession chains) was intentionally left behind and is recoverable from the tag described below.

Install:
```bash
git clone https://github.com/issdandavis/scbe-docs-archive.git
```

**Note:** only a curated subset was migrated. The complete pre-split `docs/` tree — including everything omitted from `scbe-docs-archive` — is preserved at tag `v-monolith-final` in this repo:

```bash
git checkout v-monolith-final
```
