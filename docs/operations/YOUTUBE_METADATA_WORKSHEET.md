# YouTube Metadata Worksheet

Use this after the video is rendered or published. The goal is to turn a good video into a searchable, correctly-packaged video without inventing claims that are not in the cut.

## Workflow

1. Watch the video once without editing.
2. Write down the core facts, not the intended story.
3. Fill every field below before touching title, description, or tags.
4. Rewrite the metadata from the worksheet.
5. Rerun `python scripts/apollo/video_review.py review-all`.

## Pass / Fail Check

- Description is at least 2-3 real sentences.
- Description matches the actual video.
- Core search phrase appears naturally.
- Brand / project context is present.
- CTA is present if needed.
- Tags are specific, not filler.
- No invented claims.
