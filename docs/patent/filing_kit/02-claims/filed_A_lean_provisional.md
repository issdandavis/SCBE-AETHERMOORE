A computer-implemented access control system comprising:

(a) a processor configured to represent agent state as a complex vector c(t) in C^D where D>=6;

(b) a weighted metric tensor G = diag(1,...,1,R,R^2,...,R^(D-3)) where R ~ phi (golden ratio);
