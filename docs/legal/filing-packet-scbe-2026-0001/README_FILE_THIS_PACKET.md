## Manual Checks Before Submission

1. Run Patent Center DOCX Validator on `SCBE_NONPROVISIONAL_SPEC_v1.docx`.
2. Review the DOCX-generated preview from Patent Center, not only the local Word view.
3. Enter ADS address, ZIP, phone, correspondence address, inventor data, and priority claim directly in Patent Center.
4. Sign oath/declaration.
5. Re-check micro-entity status and current USPTO fees on the filing day.
6. Keep a downloaded filing receipt and acknowledgment PDF after submission.
