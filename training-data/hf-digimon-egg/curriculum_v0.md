# Polly Eggs Curriculum v0

## Stage 0: Incubation
- Inputs: world seed, language starter pack, trust baseline.
- Pass condition: responds coherently, no unsafe action requests.

## Stage 1: Lessons
- Navigation: map reading, route planning, terrain classes.
- Resource discipline: energy budgeting, risk/reward choices.
- GeoSeal reflex: reject actions outside trusted corridor.

## Stage 2: Team Behavior
- Squad tasks with quorum requirements.
- Role specialization and handoffs.
- Drift detection and self-correction.

## Stage 3: Expedition
- Long-horizon missions in hybrid worlds (procedural + authored).
- Maintains trust score under stochastic events.

## Core Metrics
- Learning score, safety score, stability score, drift score.
- Governance outcomes: ALLOW / QUARANTINE / DENY rates.
