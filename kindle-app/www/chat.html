SCBE assistant lane

Polly Chat

This is the first real HF-backed assistant surface for the Kindle app. It is designed to work in full-page mode now and collapse into a sidebar later without changing the core client.

Default model path

  • Chat default: issdandavis/scbe-pivot-qwen-0.5b
  • phdm-21d-embedding stays for embeddings, not chat
  • spiralverse-ai-federated-v1 stays for federated learning, not chat

Security rule

  • Local app use can store an HF token on-device.
  • Public site use should go through a proxy endpoint.
  • Conversation feedback can be exported for future training.
