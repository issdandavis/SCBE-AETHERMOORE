## Long Description (4000 chars)
AetherCode is a governed AI development workspace that puts multiple AI models
at your fingertips — all running through a 14-layer security pipeline.

**Four Modes in One App:**
- **Chat** — Converse with AI models (Groq, Google AI, Claude, xAI, HuggingFace, Ollama)
- **Code** — Write and preview code with split-view editor
- **Research** — AI-powered research with source tracking
- **Browse** — AI-guided web browsing and data extraction

**Key Features:**
- Bring Your Own Keys (BYOK) — use free models or your own API keys
- Governance-aware — every response passes through safety scoring
- Multi-model routing via OctoArmor (11+ AI providers)
- Training data export — every conversation generates SFT pairs
- Voice input support
- Offline-capable — cached UI works without internet
- Dark theme optimized for Kindle Fire displays

**Free Forever:**
AetherCode is completely free. You pay AI providers directly through your own
API keys. Free models (Ollama, HuggingFace, GitHub Models) work out of the box.

**Built with SCBE-AETHERMOORE:**
Powered by the 14-layer Symphonic Cipher Blockchain Engine — the same governance
framework used in enterprise AI safety deployments.
