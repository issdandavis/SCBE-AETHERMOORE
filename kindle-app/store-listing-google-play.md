## Full Description (4000 chars max)
AetherCode is a free, governed AI development workspace that puts 9+ AI models at your fingertips — all running through a 14-layer security pipeline.

**AI Round Table — Multi-Model Chat**
Ask one model or broadcast to all. Groq, Google Gemini, Claude, Grok, HuggingFace, GitHub Models, Ollama, and more sit at your round table. Each model has a Sacred Tongue role (Intent Analyst, Security Auditor, Lead Architect, etc.) and contributes a unique perspective.

**Five Interaction Modes:**
- **Deal** — Broadcast your prompt to all models simultaneously
- **Deliberate** — Get responses from all models, then synthesize consensus
- **Research** — AI-powered web research with evidence packs and citations
- **Relay** — Chain models together, each building on the last response
- **Debate** — Pit two models against each other, then merge the best ideas

**Bring Your Own Keys (BYOK):**
AetherCode is completely free. You bring your own API keys from any provider:
- Free options: Groq, HuggingFace Inference, GitHub Models, Ollama (local)
- Premium options: OpenAI, Anthropic Claude, Google AI, xAI Grok

Your API keys are stored locally on your device and NEVER sent to our servers.

**Key Features:**
- 9 AI provider seats with automatic proxy routing
- Governance-scored responses (14-layer Symphonic Cipher pipeline)
- Shared code editor visible to all models
- Training data export — every conversation generates SFT pairs
- Offline-capable — cached UI works without internet
- Dark theme optimized for mobile and tablet displays
- Cross-talk sync between devices

**Built with SCBE-AETHERMOORE:**
Powered by the 14-layer Symphonic Cipher Blockchain Engine — an open-source AI safety framework using hyperbolic geometry to make adversarial intent exponentially more expensive.
