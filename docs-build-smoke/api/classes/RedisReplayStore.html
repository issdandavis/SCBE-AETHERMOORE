    Class RedisReplayStore

    Redis-based replay store for distributed deployments. Requires ioredis or compatible Redis client.

    Usage: import Redis from 'ioredis'; const redis = new Redis(process.env.REDIS_URL); const store = new RedisReplayStore(redis, { prefix: 'scbe:replay:' });
