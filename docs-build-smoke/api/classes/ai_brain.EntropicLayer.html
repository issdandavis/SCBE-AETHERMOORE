    computeExpansionVolume

    • computeExpansionVolume(position: number[]): number

      Compute approximate hyperbolic volume for a state position.

      For a point at radius r in the Poincare ball in d dimensions, the hyperbolic volume of the ball of that radius is approximately: V ~ (pi^(d/2) * r^d / Gamma(d/2+1)) * exp((d-1) * r)

      For 6D (our Sacred Tongues manifold): V ~ (pi^3 * r^6 / 6) * exp(5r)
