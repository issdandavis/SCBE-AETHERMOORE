Quick start

Three commands. No account needed.

1

Install

Add the governance pipeline to any Node.js or Python project.

npm install scbe-aethermoore

pip install scbe-aethermoore
2

Run a governance scan

Pass any AI intent through the 14-layer pipeline and get a governed decision back.

import { runPipeline } from 'scbe-aethermoore';

const result = await runPipeline('analyze changed files for security');
console.log(result.plan.policy.decision); // → ALLOW
3

Read the receipt

Every decision produces a signed, auditable JSON receipt. No black box.

{
"decision": "ALLOW",
"reason": "intent within governed scope",
"hashes": { "plan_sha256": "a3f9c2...", "intent_sha256": "7b01d4..." },
"layers": { "harmonic_score": 0.94, "hyperbolic_distance": 0.12 }
